# Web Tools - Memory

## Outils Web Disponibles

### 1. web_search
- **Fonction** : Recherche web via Brave Search API
- **Usage** : Rechercher des informations sur le web avec filtres (date, langue, région)
- **Output** : Titres, URLs et snippets

### 2. web_fetch
- **Fonction** : Extraction de contenu depuis une URL
- **Usage** : Récupérer le contenu texte/markdown d'une page web spécifique
- **Output** : Texte brut ou markdown

### 3. browser
- **Fonction** : Navigation Chromium complète
- **Usage** : Contrôler Chrome (clics, screenshots, formulaires, navigation)
- **Output** : Captures d'écran, snapshots de page, actions interactives

### 4. curl (via exec)
- **Fonction** : Requêtes HTTP via commande curl
- **Usage** : Appels API avec headers personnalisés, méthodes HTTP (GET, POST, etc.)
- **Output** : Réponses HTTP brutes
